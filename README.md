# 消息中转小工具APP


满足特定需求用户，手机不在身边或在办公场所下，不方便查看手机上的推送通知；


### 功能
* 收纳手机推送通知，自动转发到邮箱；


### 对如下应用提供收纳、转发服务
* QQ、微信、手机短信；

##### 1. 参数配置

> 个人邮箱须要开启SMTP功能；
